Detailed Description

Clase abstracta que representa un pokémon genérico.
Invariante de clase: Pokemon distinto de null.
Define atributos y métodos comunes a todos los Pokemon.
Implementa las interfaces interfaces.IHostil, interfaces.IValuable, interfaces.IClasificable, interfaces.IHechizable y utiliza java.lang.Cloneable para permitir la clonación de objetos.
